# Credit_Card_Customer_Segmentation

This project uses K-Means clustering to segment credit card customers based on their usage behavior.

## Dataset

The dataset used is `CC_GENERAL.csv`.

## Methods Used

- Data cleaning
- Mean imputation
- Exploratory data analysis
- Feature scaling using StandardScaler
- K-Means clustering
- Elbow method
- Silhouette score
- PCA visualization

## Goal

The goal is to identify customer segments based on credit card behavior and understand how these segments can support marketing strategies.
